# OPC UA Client for CNC Systems (C#)

Профессиональное клиентское приложение для работы с ЧПУ "АксиОМА Контрол" по протоколу OPC UA.

### Основные функции:
* **Подключение**: Асинхронное соединение с сервером
* **Подписки (Subscriptions)**: Мониторинг переменных (время работы, координаты) в реальном времени через событийную модель
* **UI**: Пользовательские элементы управления (User Controls) для визуализации данных ЧПУ

### Технологии:
* .NET Framework / WinForms.
* Библиотека OPC Foundation (`Opc.Ua.Client`).
